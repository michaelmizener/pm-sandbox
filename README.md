# Launchpad PM Sandbox

A fictional SaaS product used as a hands-on learning environment for Claude Code.

**Product:** Launchpad — an internal launch coordination tool for product and GTM teams.

## Structure

| File/Folder | Purpose |
|---|---|
| `prd/` | Product requirements documents |
| `roadmap.md` | Quarterly roadmap |
| `changelog.md` | Release notes |
| `decisions/` | Architecture & product decision records |
| `research/` | User research summaries |

## How to use this with Claude Code

- Ask Claude to improve or critique any spec
- Ask Claude to create a new GitHub issue from a PRD section
- Ask Claude to compare two versions of a feature spec
- Ask Claude to draft a changelog entry from a PRD
